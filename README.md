# autoTweet
Tweet From Terminal

Its python program that helps you tweet directly from the terminal.

Pre-Requisites:																										
1. Generate OAuth authentication token and keys                   								
2. Install python :P

Files:																										
1. autoTweet.py --- Its the python program that has to be run from the terminal window							
2. twitterParams.txt -- It consists of all the twitter parameters you must have before running the python code.


How to Run:																											
1. Open twitterParams.txt and set the text in the 'status' column which is the text you need to tweet. 					
2. Run "python autoTweet.py" in your terminal.
